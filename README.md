# 🚀 Nexus ERP

Sistema ERP completo desenvolvido em **Java 21** com **Spring Boot**, criado com foco em arquitetura limpa, boas práticas de desenvolvimento e escalabilidade.

## 📋 Sobre o projeto

O Nexus ERP tem como objetivo centralizar e automatizar processos empresariais como:

- 👤 Gestão de Usuários
- 📦 Controle de Estoque
- 🛒 Compras
- 💰 Financeiro
- 🚚 Logística
- 👥 Recursos Humanos
- 📊 Dashboard Gerencial
- 🔐 Controle de Permissões
- 📋 Auditoria
- 📈 Relatórios

## 🛠️ Tecnologias

- Java 21
- Spring Boot
- Spring Security
- PostgreSQL
- Flyway
- Maven
- Docker (em breve)
- Swagger/OpenAPI (em breve)
- JUnit 5
